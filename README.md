
# 🌦️ Weather Prediction System — MLOps Pipeline

## 📌 Overview
This project is an end-to-end Machine Learning and MLOps pipeline designed to predict weather conditions for the next few hours. It automates the complete workflow from data ingestion to model deployment and monitoring.

---

## ✨ Features
- 📊 Data ingestion and preprocessing pipeline
- 🤖 Machine Learning model for weather prediction
- 🔁 Automated training and forecasting pipeline
- 📈 Experiment tracking and model versioning using MLflow
- ⚙️ Deployment-ready pipeline with CI/CD workflows
- 📦 Containerized application for scalability

---

## 🏗️ Architecture
- **Data Processing:** Pandas, NumPy  
- **Modeling:** LightGBM / ML algorithms  
- **Experiment Tracking:** MLflow  
- **Deployment:** Docker  
- **Automation:** CI/CD pipelines  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn / LightGBM  
- MLflow  
- Docker  
- GitHub Actions
